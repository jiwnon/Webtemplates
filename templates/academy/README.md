# 학원 템플릿

학원·입시 학원용 랜딩 페이지입니다. 수강 신청 전환, 모바일·카카오톡 상담을 강조했습니다.

## 포함 내용

- **학원 소개** · **강사 프로필**
- **시간표** · **수강료 안내**
- **수강 신청 폼** (바로 전환용, `/api/contacts` 연동 가능)
- **학부모 후기**
- **네이버 예약** 링크 (연동 후 URL 교체)
- **오시는 길** + 지도 영역 (카카오맵/네이버맵 연동)
- **카카오톡 상담** 플로팅 버튼 (크게, 모바일 최적화)

## 커스터마이징

| 항목 | 위치 |
|------|------|
| 학원명 | `.site-title`, `hero h1`, footer |
| 강사 정보 | `#teachers` 카드 |
| 시간표 | `#schedule` 테이블 |
| 수강료 | `#tuition` 카드 |
| 카카오톡 링크 | `.kakao-float` 의 `href` (pf.kakao.com 주소) |
| 네이버 예약 | `#naver-booking` 내 버튼 `href` |
| 지도 | `.map-placeholder` 를 카카오/네이버 맵 iframe 또는 API로 교체 |

## API 연동

수강 신청 폼은 `POST /api/contacts` 로 전송됩니다. D1 설정 후 `template: 'academy'` 로 저장됩니다.  
백엔드가 없으면 `script.js` 에서 `fetch` 실패 시 "전화로 문의해 주세요" 안내만 나오도록 되어 있습니다.

## 모바일

뷰포트·터치 영역·safe-area 를 고려해 두었습니다. 카카오 버튼은 고정 플로팅으로 항상 보이게 했습니다.
