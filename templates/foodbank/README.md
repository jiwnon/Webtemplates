# 푸드뱅크 템플릿

기업·개인 기부와 수혜·자원봉사를 연결하는 푸드뱅크 전용 웹사이트 템플릿입니다.

## 기술 스택

- HTML, CSS, JavaScript
- Tailwind CSS (CDN)
- Chart.js (월별 기부 차트)
- 반응형 · Cloudflare Pages 배포 가능

## 디자인

- 그린(나눔·신선함) + 블루(신뢰) 계열
- 신뢰감·전문성 + 따뜻하고 깔끔한 톤

## 파일 구조

| 파일 | 설명 |
|------|------|
| index.html | 메인 (히어로, 기부 탭, 투명성, 후원 기업, 갤러리, 참여 방법, 공지, 푸터) |
| about.html | 소개 (설립 목적, 주요 사업, 연혁, 조직도) |
| donate.html | 기부 안내 상세 (물품/금전 절차, 세액 공제, FAQ) |
| volunteer.html | 자원봉사 (활동 소개, 신청 방법, 일정) |
| beneficiary.html | 수혜 안내 (자격, 신청, 배분 일정) |
| news.html | 알림마당 (공지·소식·보도) |
| script.js | 통계 카운트업, 탭, 모달, 차트, 갤러리 필터, 후원 기업 팝업 |

## 필수 섹션 (메인)

1. 히어로: 슬로건, 실시간 통계 3개(기부량/수혜가구/참여기업), CTA
2. 기부 안내: 탭(기업/개인/물품/금전), 기부 신청 모달
3. 투명성: 월별 차트, 배분 현황, 재정 보고서 다운로드
4. 후원 감사: 기업 로고 그리드, 클릭 시 상세 팝업
5. 활동 갤러리: 카테고리 필터
6. 참여 방법: 기부/봉사/도움받기 3카드
7. 공지사항/소식: 최근 글 + 더보기
8. 푸터: 연락처, SNS, 사업자정보, 개인정보처리방침

## 더미 데이터

- 기관명: 행복나눔 푸드뱅크
- 통계: 15톤, 500가구, 30기업
- 후원 기업: 삼성, LG, 롯데, CJ placeholder

## 커스터마이징

- 색상: Tailwind `primary`, `trust` 또는 유틸 클래스 수정
- 통계: `[data-count]` 값 및 script.js 카운트업
- 차트: script.js Chart.js 데이터 수정
- 기부 폼: 모달 내 form을 API(/api/contacts 등)와 연동
