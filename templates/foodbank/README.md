# 푸드뱅크 템플릿 (동네형)

**동네 ○○구 푸드뱅크** 느낌의 단일 페이지 템플릿입니다.  
사랑의열매 같은 대형 단체가 아니라, 친근하고 따뜻한 현장감·일상 업데이트를 강조했습니다.

## 기술 스택

- HTML, CSS, JavaScript
- Tailwind CSS (CDN)
- 반응형 · 모바일 우선
- Cloudflare Pages / GitHub Pages 배포 가능

## 디자인

- **색상**: 따뜻한 오렌지/연두 계열 (지역아동센터 템플릿과 비슷한 톤)
- **폰트**: Noto Sans KR
- **사진**: 크고 많이 (현장감)
- **버튼 문구**: "같이해요", "참여할래요" 등 친근한 표현

## 필수 섹션 (순서)

1. **히어로** – 배분 현장 사진 배경, "○○구와 함께하는 따뜻한 나눔", 통계 3개 (이번 달 기부량, 배분 가구, 동네 기업)
2. **우리 소개** – 짧은 소개, 센터장/직원 사진 + 인사말, 연락처 크게 (전화, 카톡)
3. **기부 방법** – 3카드 (기업 / 개인 / 물품), 각 "신청하기"
4. **이번 주 활동** – 블로그 느낌, 날짜+사진+짧은 글 3~4개
5. **현장 사진 갤러리** – 그리드, 날짜 태그 (배분/봉사/기부)
6. **함께하는 동네 기업** – ○○슈퍼, △△빵집 스타일 로고/이름 그리드
7. **참여 안내** – 기부/봉사 신청 폼, 수혜 문의, 버튼 "같이해요" "참여할래요"
8. **찾아오시는 길** – 주소, 전화, 지도, 운영 시간

## 파일 구조

```
templates/foodbank/
├── index.html   # 단일 페이지 (위 8개 섹션)
├── script.js    # 모바일 메뉴, fade-in, 모달, 폼
├── images/      # 실제 사진 넣을 폴더 (placeholder)
└── README.md    # 이 파일
```
