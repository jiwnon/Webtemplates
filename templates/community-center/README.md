# 지역아동센터 템플릿

지역아동센터·아동복지시설용 단일 페이지 템플릿입니다. Tailwind CSS CDN 사용, 모바일 우선 반응형입니다.

## 기술 스택

- HTML, CSS, JavaScript
- Tailwind CSS (CDN)
- 반응형 · 모바일 우선
- GitHub Pages / Cloudflare Pages 배포 가능

## 포함 섹션

1. 히어로 (대표 이미지, 센터명, CTA: 후원하기/문의하기)
2. 센터 소개 (미션, 연혁, 시설 사진)
3. 프로그램 안내 (방과후 돌봄, 학습지도, 급식, 특별활동, 문화체험 등 카드)
4. 선생님 소개 (그리드)
5. 활동 갤러리 (그리드 + 클릭 시 라이트박스)
6. 후원 안내 (계좌, 후원자 명단 영역)
7. 오시는 길 (주소, 지도 placeholder, 연락처)
8. 문의하기 (이름, 연락처, 내용 폼 — 전송 로직은 별도 연동)
9. 푸터 (센터 정보, SNS, 저작권)

## 커스터마이징

- **센터명**: `행복지역아동센터` 검색 후 교체
- **색상**: `tailwind.config` 의 `warm`, `soft` 색상 또는 Tailwind 클래스 수정
- **이미지**: 히어로·시설·갤러리 `src` 를 실제 사진 URL로 교체
- **후원 계좌**: `#donate` 섹션 계좌번호·예금주 수정
- **지도**: 오시는 길의 placeholder 를 카카오맵/구글맵 iframe 으로 교체
- **문의 폼**: `script.js` 또는 백엔드에서 `/api/contacts` 등으로 전송 연동

## 더미 데이터

- 센터명: 행복지역아동센터
- placeholder 이미지: Unsplash (실제 사용 시 교체)
- 연락처·주소·계좌: 예시 값
